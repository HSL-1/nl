# Nanjiang Reservoir Flood Control Dispatch Platform

A flood control engineering dispatch platform based on Muskingum method and Xin'Anjiang model.

## Modules
- Data Overview
- Thiessen Polygon Analysis
- Xin'Anjiang Model
- Inflow Reverse Calculation
- Muskingum Flood Routing
- Reservoir Routing
- Dispatch Management
- Real-time Simulation

## Deploy on Streamlit Cloud
1. Fork this repo
2. Go to https://streamlit.io/cloud
3. Create new app from this repo
4. Set main file to app.py
